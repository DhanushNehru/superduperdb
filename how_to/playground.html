Download, install and setup SuperDuperDB#

Welcome to the SuperDuperDB Playground!

For demo purposes, the community edition of MongoDB is used. This should take less than a minute

[1]:

!apt install wget curl gnupg2 software-properties-common apt-transport-https ca-certificates lsb-release

!echo "deb http://security.ubuntu.com/ubuntu focal-security main" | sudo tee /etc/apt/sources.list.d/focal-security.list
!sudo apt-get update
!sudo apt-get install libssl1.1

!apt install mongodb-org
